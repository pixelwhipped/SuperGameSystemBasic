Allows groups of statements to be executed when conditions are met.
IF (expr) THEN
   ...Statements...
ELSEIF (expr) THEN
   ...Statements...
ELSE
   ...Statements...
ENDIF
 
example
Let x = 2
IF x = 1 THEN
PRINT("x is 1")
ELSEIF x = 2 THEN
PRINT("x is 2")
ELSE
PRINT("x must be something else")
ENDIF
 
output x is 2
